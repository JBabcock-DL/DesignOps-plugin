## Step 13 — Write CSS token file

### 13a — Resolve output path

### 13b — Generate and write the CSS file

**Data references:**
- [`../data/primitives-baseline.json`](../data/primitives-baseline.json) — spacing, radius, elevation, typeface
- [`../data/theme-aliases.json`](../data/theme-aliases.json) — Theme variable WEB codeSyntax (used to build `--color-*` custom properties)
- [`../data/typography-slots.json`](../data/typography-slots.json) — slot base values + semantic alias map
- [`../data/layout-effects.json`](../data/layout-effects.json) — spacing/radius aliases + shadow custom properties
- [`../data/tokens-css-shadcn.json`](../data/tokens-css-shadcn.json) — shadcn/ui legacy alias map + body-variant decoration helpers

**File structure (emit sections in this order):**

```
/* Header — project name + "DO NOT EDIT MANUALLY / generated by /create-design-system" */

:root {
  /* Primitives — Color ramps (5 ramps × 11 stops, --color-{ramp}-{stop}: {hex}) */
  /* Primitives — Spacing (--space-{N}: {base * multiplier}px) */
  /* Primitives — Corner radius (--corner-{name}: {value}px) */
  /* Primitives — Elevation (--elevation-{N}: {value}) */
  /* Primitives — Typeface (--typeface-display: {family}; --typeface-body: {family}) */
}

:root {
  /* Layout — Spacing aliases (--space-xs … --space-4xl: var(--space-{N})) */
  /* Layout — Radius aliases (--radius-none … --radius-full: var(--corner-{name})) */
  /* shadcn/ui base radius — --radius: var(--radius-md) */
}

:root, [data-theme="light"] {
  /* Theme Light — --color-* custom properties (WEB codeSyntax from theme-aliases.json rows[*]) */
  /* Effects Light — --shadow-color: rgba(0,0,0,0.10); blur aliases as calc(var(--elevation-N)*1px) */
  /* shadcn/ui legacy aliases — from tokens-css-shadcn.json lightAndDark (same values in Light context) */
}

[data-theme="dark"] {
  /* Theme Dark — same --color-* names; resolved from dark column in theme-aliases.json */
  /* Effects Dark — --shadow-color: rgba(0,0,0,0.30) */
  /* shadcn/ui legacy aliases */
}

@media (prefers-color-scheme: dark) {
  :root:not([data-theme="light"]) {
    /* Duplicate of [data-theme="dark"] block so system preference works without JS */
  }
}

:root {
  /* Typography mode 100 — 15 slots × 4 properties (--{slot-kebab}-{property-kebab}: {value}) */
  /* Body variants — 12 slots aliasing base body (size/line/family) + weight/decoration overrides */
  /* Body-variant decoration helpers from tokens-css-shadcn.json bodyVariantDecorationHelpers */
  /* Semantic aliases from typography-slots.json tokensCssSemanticAliases (--text-h1-* etc.) */
}

[data-font-scale="85"]  { /* 15 slots × font-size + line-height computed for 85  */ }
[data-font-scale="110"] { /* …110 */ }
[data-font-scale="120"] { /* …120 */ }
[data-font-scale="130"] { /* …130 */ }
[data-font-scale="150"] { /* …150 */ }
[data-font-scale="175"] { /* …175 */ }
[data-font-scale="200"] { /* …200 */ }
/* Semantic --text-h1-* etc. aliases reference the role tokens and auto-track [data-font-scale] — do NOT repeat computed values in scale blocks. */
```

**Key rules:**
- Compute all scale-mode font-size/line-height values using the Step 7 formula before writing — never write `{computed}` placeholders.
- `@media (prefers-color-scheme: dark)` block duplicates the dark values. This duplication is intentional.
- The `--radius` shorthand for shadcn = `var(--radius-md)` — emit once in the Layout block.
